# AllOfSoccer App Store Readiness Plan

## Product Goal
- 축구/풋살 팀이 매주 상대팀 매칭과 용병 모집을 모바일에서 빠르고 일관되게 처리한다.
- 앱 내 메신저 대신 전화/문자 중심 연락으로 매칭 성사율을 높인다.

## UX Principles
- 팀 매치 / 용병 모집 화면은 동일한 정보 구조와 카드 형태를 유지한다.
- 핵심 액션(필터, 상세 보기, 신청/등록)은 2탭 이내에서 완료 가능해야 한다.
- 상태(모집중/완료, 선출 포함 여부, 참가비, 실력 요구)는 리스트에서 바로 보이게 한다.

## Release Phases
1. 안정화 (현재)
- 리스트 페이징 과호출/429 방지
- 작성/상세/필터 핵심 버그 제거
- 마이페이지 API 연동(프로필, 관심글, 내가쓴글, 내 팀)

2. 기능 완성
- 팀 가입 승인대기 플로우
- 실제 매칭 검증 기반 리뷰 권한
- 팀/매칭 평점 UI 연결

3. 운영 준비
- 로깅/크래시/분석(Sentry/Firebase)
- QA 회귀 테스트 체크리스트
- 앱스토어 메타데이터/스크린샷/개인정보 문서

## Team Workstreams
- iOS: 공통 컴포넌트화, 성능/접근성, 릴리즈 빌드 안정성
- Server: 필터/권한/리뷰 검증, rate limit 운영값 분리
- QA: 시나리오 기반 테스트(필터/작성/신청/문의/마이페이지)
- 기획/디자인: 정보 우선순위, 문구 통일, 상태 표현 규칙

## Definition of Done (MVP)
- 팀 매치/용병 모집 리스트-상세-등록 플로우가 실서버 API 기준으로 안정 동작
- 마이페이지(프로필, 내 팀, 관심글, 내가 쓴 글, 문의하기)가 끊김 없이 동작
- 크래시/치명 버그 0, 회귀 테스트 통과, TestFlight 배포 가능 상태
